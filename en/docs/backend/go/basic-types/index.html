Go иҜӯиЁҖеҹәзЎҖзұ»еһӢ

Go иҜӯиЁҖд№ӢжүҖд»ҘзңӢиө·жқҘеҹәзЎҖзұ»еһӢпјҲBasic Typesпјүзү№еҲ«еӨҡпјҢжҳҜеӣ дёәе®ғжҳҜдёҖй—Ёејәзұ»еһӢдё”еҜ№еҶ…еӯҳжҺ§еҲ¶еҫҲзІҫзЎ®зҡ„иҜӯиЁҖгҖӮ

дҪҶе…¶е®һдҪ дёҚйңҖиҰҒжӯ»и®°зЎ¬иғҢжҜҸдёҖдёӘгҖӮжҲ‘д»¬еҸҜд»ҘжҠҠе®ғд»¬еҪ’зәідёәеӣӣеӨ§зұ»гҖӮеҸӘиҰҒжҺҢжҸЎдәҶж ёеҝғйҖ»иҫ‘пјҢеү©дёӢзҡ„йғҪжҳҜеҸҳдҪ“гҖӮ

1. еёғе°”еһӢ (Boolean)вҖӢ

жңҖз®ҖеҚ•зҡ„дёҖз§ҚпјҢеҸӘжңүзңҹе’ҢеҒҮгҖӮ

  • зұ»еһӢе…ій”®еӯ—пјҡ bool
  • еҖјпјҡ true жҲ– false
  • жіЁж„Ҹпјҡ Go иҜӯиЁҖдёӯпјҢеёғе°”еҖјдёҚиғҪе’Ң 0 жҲ– 1 зӣёдә’иҪ¬жҚўпјҲиҝҷе’Ң C иҜӯиЁҖжҲ– Python дёҚеҗҢпјүгҖӮ

2. еӯ—з¬ҰдёІ (String)вҖӢ

Go зҡ„еӯ—з¬ҰдёІеӨ„зҗҶйқһеёёејәеӨ§пјҢй»ҳи®Өж”ҜжҢҒ UTF-8пјҲд№ҹе°ұжҳҜиҜҙзӣҙжҺҘж”ҜжҢҒдёӯж–ҮпјүгҖӮ

  • зұ»еһӢе…ій”®еӯ—пјҡ string
  • зү№зӮ№пјҡ дёҖж—ҰеҲӣе»әпјҢеҶ…е®№дёҚеҸҜеҸҳгҖӮ

3. ж•°еӯ—зұ»еһӢ (Numeric Types)вҖӢ

иҝҷжҳҜи®©дҪ ж„ҹи§ү"еӨҡ"зҡ„зҪӘйӯҒзҘёйҰ–гҖӮGo жҠҠж•°еӯ—жӢҶеҲҶеҫ—еҫҲз»ҶпјҢжҳҜдёәдәҶеңЁзі»з»ҹзј–зЁӢдёӯиҠӮзңҒеҶ…еӯҳгҖӮ

A. ж•ҙж•° (Integers)вҖӢ

еҲҶдёәжңүз¬ҰеҸ·пјҲжӯЈиҙҹж•°пјүе’Ңж— з¬ҰеҸ·пјҲеҸӘжңүжӯЈж•°пјҢUnsignedпјүгҖӮ

зұ»еһӢжҸҸиҝ°еёёз”ЁеңәжҷҜ
intжңҖеёёз”ЁгҖӮж №жҚ®зі»з»ҹжҳҜ32дҪҚиҝҳжҳҜ64дҪҚиҮӘеҠЁеҸҳеҢ–гҖӮ90% зҡ„жғ…еҶөз”Ёе®ғ
int8, int16, int32, int64еӣәе®ҡеӨ§е°Ҹзҡ„ж•ҙж•°пјҲ8дҪҚеҲ°64дҪҚпјүгҖӮеӨ„зҗҶеә•еұӮеҚҸи®®гҖҒж–Үд»¶ж јејҸжҲ–еӨ§ж•°
uintж— з¬ҰеҸ·ж•ҙж•°пјҢзұ»дјј int дҪҶдёҚиғҪдёәиҙҹгҖӮдҪҚиҝҗз®—гҖҒзү№е®ҡз®—жі•
uint8, uint16, uint32, uint64еӣәе®ҡеӨ§е°Ҹзҡ„ж— з¬ҰеҸ·ж•ҙж•°гҖӮеҗҢдёҠ

B. жө®зӮ№ж•° (Floating Point)вҖӢ

д№ҹе°ұжҳҜе°Ҹж•°гҖӮ

  • float32: зІҫеәҰиҫғдҪҺгҖӮ
  • float64: жңҖеёёз”ЁгҖӮзІҫеәҰй«ҳпјҢGo зҡ„ж•°еӯҰеә“й»ҳи®ӨйғҪз”ЁиҝҷдёӘгҖӮ

C. еӨҚж•° (Complex)вҖӢ

з”ЁдәҺз§‘еӯҰи®Ўз®—пјҢдёҖиҲ¬дёҡеҠЎејҖеҸ‘еҫҲе°‘з”ЁеҲ°гҖӮ

  • complex64, complex128

4. дёӨдёӘжһҒе…¶йҮҚиҰҒзҡ„"еҲ«еҗҚ"вҖӢ

Go жңүдёӨдёӘзү№ж®Ҡзҡ„зұ»еһӢеҗҚеӯ—пјҢз»ҸеёёзңӢеҲ°пјҢдёҖе®ҡиҰҒи®°дҪҸе®ғд»¬д»ЈиЎЁд»Җд№Ҳпјҡ

byteвҖӢ

  • е…¶е®һе°ұжҳҜ uint8гҖӮ
  • з”ЁйҖ”пјҡ з”ЁжқҘеӨ„зҗҶеҺҹе§Ӣж•°жҚ®пјҲеҰӮж–Үд»¶жөҒгҖҒзҪ‘з»ңдј иҫ“пјүгҖҒASCII еӯ—з¬ҰгҖӮ

runeвҖӢ

  • е…¶е®һе°ұжҳҜ int32гҖӮ
  • з”ЁйҖ”пјҡ д»ЈиЎЁдёҖдёӘ Unicode еӯ—з¬ҰгҖӮ
  • дёәд»Җд№ҲйҮҚиҰҒпјҹ еӣ дёәдёӯж–Үеӯ—з¬ҰеңЁ UTF-8 дёӯеҚ  3 дёӘеӯ—иҠӮпјҢз”Ё byte еӯҳдёҚдёӢпјҢеҝ…йЎ»з”Ё rune жқҘеӯҳеҚ•зӢ¬зҡ„дёҖдёӘжұүеӯ—гҖӮ

жһҒз®Җз”ҹеӯҳжҢҮеҚ—пјҡжҲ‘иҜҘз”Ёе“ӘдёӘпјҹвҖӢ

иҷҪз„¶жңүиҝҷд№ҲеӨҡзұ»еһӢпјҢдҪҶеңЁжҷ®йҖҡзҡ„дёҡеҠЎд»Јз ҒпјҲWeb ејҖеҸ‘гҖҒе·Ҙе…·и„ҡжң¬пјүдёӯпјҢдҪ еҸӘйңҖиҰҒи®°дҪҸд»ҘдёӢ 4 дёӘй»„йҮ‘ж ҮеҮҶ е°ұеӨҹдәҶпјҡ

  1. жҳҜж•ҙж•°пјҹз”Ё intгҖӮ
  2. жҳҜе°Ҹж•°пјҹз”Ё float64гҖӮ
  3. жҳҜзңҹеҒҮпјҹз”Ё boolгҖӮ
  4. жҳҜж–Үеӯ—пјҹз”Ё stringгҖӮ

еҸӘжңүеҪ“дҪ жҳҺзЎ®зҹҘйҒ“"жҲ‘йңҖиҰҒеӨ„зҗҶдәҢиҝӣеҲ¶жөҒ"жҲ–иҖ…"жҲ‘иҰҒеҒҡдёҖдёӘеҜ№еҶ…еӯҳжһҒе…¶ж•Ҹж„ҹзҡ„еөҢе…ҘејҸзЁӢеәҸ"ж—¶пјҢжүҚеҺ»иҖғиҷ‘йӮЈдәӣ int8гҖҒuint32 д№Ӣзұ»зҡ„зұ»еһӢгҖӮ

зү№еҲ«иЎҘе……пјҡйӣ¶еҖј (Zero Value)вҖӢ

Go жңүдёҖдёӘеҫҲжЈ’зҡ„зү№жҖ§пјҡжүҖжңүеҸҳйҮҸиў«еЈ°жҳҺж—¶пјҢйғҪдјҡиҮӘеҠЁжңүдёҖдёӘй»ҳи®ӨеҖјпјҲдёҚдјҡеғҸ C иҜӯиЁҖйӮЈж ·жҳҜйҡҸжңәеһғеңҫеҖјпјүгҖӮ

  • int / float й»ҳи®Өдёә 0
  • bool й»ҳи®Өдёә false
  • string й»ҳи®Өдёә "" (з©әеӯ—з¬ҰдёІ)

д»Јз ҒзӨәдҫӢвҖӢ

package main

import "fmt"

func main() {
    var a int      // й»ҳи®Өдёә 0
    var b string   // й»ҳи®Өдёә ""
    var c bool     // й»ҳи®Өдёә false

    // иҝҷйҮҢз”ЁеҲ°дәҶ rune (int32) жқҘжү“еҚ°жұүеӯ—
    var d rune = 'еҘҪ'

    fmt.Printf("a: %d, b: %q, c: %v, d: %c\n", a, b, c, d)
}
