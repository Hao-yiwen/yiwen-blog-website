TouchableWithoutFeedback

<TouchableHighlight onPress={() => {}} >
  <Text>Home</Text>
</TouchableHighlight>

为什么这段代码中，在TouchableHighlight被按下时候有灰色椭圆突出显示文字？​

这是Text中的suppressHighlighting属性导致的，只针对ios有效，默认是有灰色椭圆效果，可添加suppressHighlighting属性来修改Text按下后默认展示效果。

<TouchableWithoutFeedback onPress={() => {}}>
    <Text suppressHighlighting>Home</Text>
</TouchableWithoutFeedback>
