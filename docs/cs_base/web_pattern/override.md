# 总结

现阶段大前端的实现中主要使用mvvm,因为大多数框架都做到了view和state的绑定。很少需要手动更新视图，除了古老的jqurey和手动操作dom需要。

## 常见的mvvm实现

- React Native + Redux 
- ios: uikit + viewmodel / compose + viewmodel
- android: view + viewmodel / swift + viewmodel
