Go еҲҮзүҮдј еҸӮпјҡдёәд»Җд№Ҳ append еӨ–йғЁзңӢдёҚеҲ°пјҹ

иҝҷдёӘй—®йўҳйқһеёёз»Ҹе…ёпјҢж ёеҝғеҺҹеӣ еҸҜд»Ҙз”ЁдёҖеҸҘиҜқжҰӮжӢ¬пјҡ

Go еҮҪж•°дј еҸӮжҳҜ"еҖјдј йҖ’"пјҢдҪ дј иҝӣеҺ»зҡ„еҲҮзүҮе…¶е®һжҳҜдёҖдёӘ"з»“жһ„дҪ“зҡ„еүҜжң¬"гҖӮ

иҷҪз„¶е®ғд»¬е…ұдә«еә•еұӮж•°з»„пјҲжҢҮй’ҲдёҖж ·пјүпјҢдҪҶе®ғд»¬жӢҘжңүеҗ„иҮӘзӢ¬з«Ӣзҡ„ LenпјҲй•ҝеәҰпјүе’Ң CapпјҲе®№йҮҸпјүгҖӮ

1. дёәд»Җд№Ҳ s[0]=x еӨ–йғЁиғҪзңӢеҲ°пјҹвҖӢ

еӣ дёәжҢҮй’ҲжҢҮеҗ‘зҡ„жҳҜеҗҢдёҖдёӘең°ж–№гҖӮ

еҪ“дҪ дј йҖ’еҲҮзүҮж—¶пјҢGo еӨҚеҲ¶дәҶйӮЈдёӘ"еҲҮзүҮеӨҙ"пјҲSliceHeaderпјүпјҡ

  • еӨ–йғЁеҸҳйҮҸпјҡ {Ptr: 0x100, Len: 3, Cap: 5}
  • еҮҪж•°еҶ…йғЁеҸҳйҮҸпјҡ {Ptr: 0x100, Len: 3, Cap: 5}

жіЁж„Ҹ Ptr йғҪжҳҜ 0x100гҖӮ еҮҪж•°жү§иЎҢ s[0] = 999 ж—¶пјҢе®ғйЎәзқҖ 0x100 иҝҷдёӘең°еқҖеҺ»дҝ®ж”№еҶ…еӯҳгҖӮ

з»“жһңпјҡ еҶ…еӯҳеҸҳдәҶгҖӮеӨ–йғЁеҸҳйҮҸд№ҹзңӢ 0x100пјҢжүҖд»Ҙе®ғзңӢеҲ°дәҶеҸҳеҢ–гҖӮ

2. дёәд»Җд№Ҳ append еӨ–йғЁзңӢдёҚеҲ°пјҹвҖӢ

иҝҷжҳҜжңҖеқ‘зҡ„ең°ж–№гҖӮappend дјҡеҜјиҮҙдёӨз§Қжғ…еҶөпјҢдҪҶж— и®әе“Әз§ҚпјҢеӨ–йғЁйғҪзңӢдёҚеҲ°гҖӮ

жғ…еҶө Aпјҡе®№йҮҸи¶іеӨҹпјҲжІЎеҸ‘з”ҹжү©е®№пјүпјҢдҪҶ"и§ҶйҮҺ"жІЎи·ҹдёҠвҖӢ

еҒҮи®ҫ Len=3, Cap=5гҖӮ дҪ  append дәҶдёҖдёӘе…ғзҙ пјҢеә•еұӮж•°з»„е…¶е®һзңҹзҡ„иў«дҝ®ж”№дәҶпјҲз¬¬4дёӘдҪҚзҪ®еҶҷе…ҘдәҶж•°жҚ®пјүгҖӮ

дҪҶжҳҜпјҒSliceHeader йҮҢзҡ„ Len еҸҳдәҶгҖӮ

  • еҮҪж•°еҶ…йғЁеҸҳйҮҸпјҡ {Ptr: 0x100, Len: 4, Cap: 5} пјҲй•ҝеәҰеҸҳжҲҗдәҶ 4пјү
  • еӨ–йғЁеҸҳйҮҸпјҡ {Ptr: 0x100, Len: 3, Cap: 5} пјҲй•ҝеәҰдҫқз„¶жҳҜ 3пјү

з»“жһңпјҡ иҷҪз„¶ж•°жҚ®еҶҷиҝӣеҺ»дәҶпјҢдҪҶеӨ–йғЁеҲҮзүҮзҡ„ Len й”Ғжӯ»еңЁ 3гҖӮжү“еҚ°зҡ„ж—¶еҖҷпјҢGo еҸӘдјҡйҒҚеҺҶеҲ° Len дёәжӯўгҖӮеӨ–йғЁеҲҮзүҮи§үеҫ—"жҲ‘еҸӘжңү 3 дёӘе…ғзҙ "пјҢжүҖд»Ҙе®ғзңӢдёҚи§Ғз¬¬ 4 дёӘе…ғзҙ гҖӮ

жғ…еҶө Bпјҡе®№йҮҸдёҚеӨҹпјҲеҸ‘з”ҹдәҶжү©е®№пјүпјҢеҪ»еә•еҲҶе®¶вҖӢ

еҒҮи®ҫ Len=3, Cap=3гҖӮ дҪ  append дәҶдёҖдёӘе…ғзҙ пјҢеҺҹжқҘзҡ„еқ‘еЎ«дёҚдёӢдәҶгҖӮ

  • еҮҪж•°еҶ…йғЁпјҡ append еҸ‘зҺ°дёҚеӨҹз”ЁпјҢејҖиҫҹдәҶдёҖеқ—ж–°еҶ…еӯҳең°еқҖ 0x888пјҢжҠҠиҖҒж•°жҚ®жӢ·иҝҮеҺ»пјҢеҶҚиҝҪеҠ ж–°ж•°жҚ®гҖӮ

  • еҶ…йғЁеҸҳйҮҸеҸҳжҲҗпјҡ{Ptr: 0x888, Len: 4, Cap: 6}

  • еӨ–йғЁеҸҳйҮҸпјҡ дҫқз„¶жҢҮеҗ‘иҖҒе®¶гҖӮ

  • еӨ–йғЁеҸҳйҮҸдҝқжҢҒпјҡ{Ptr: 0x100, Len: 3, Cap: 3}

з»“жһңпјҡ еҶ…йғЁеҸҳйҮҸе·Із»Ҹжҗ¬е®¶дәҶпјҢеӨ–йғЁеҸҳйҮҸиҝҳеңЁзңӢиҖҒжҲҝеӯҗгҖӮеҪ»еә•ж— е…ідәҶгҖӮ

еӣҫи§ЈеҜ№жҜ”вҖӢ

жҲ‘д»¬еҸҜд»ҘжҠҠеҲҮзүҮжғіиұЎжҲҗдёҖеј "иҙӯзү©жё…еҚ•"пјҡ

жё…еҚ•еҶ…е®№пјҡ

  1. д»“еә“ең°еқҖпјҡAеҢә1еҸ·жһ¶
  2. иҰҒжӢҝеҮ дёӘе•Ҷе“Ғпјҡ3дёӘ (Len)

дј йҖ’з»ҷеҮҪж•°ж—¶пјҢдҪ еӨҚеҚ°дәҶдёҖд»Ҫжё…еҚ•з»ҷеҮҪж•°гҖӮ

  • ж“ҚдҪң 1пјҡдҝ®ж”№е•Ҷе“Ғ (s[0] = x) еҮҪж•°жӢҝзқҖеӨҚеҚ°д»¶пјҢеҺ» AеҢә1еҸ·жһ¶пјҢжҠҠз¬¬дёҖдёӘиӢ№жһңжҚўжҲҗдәҶйҰҷи•үгҖӮ дҪ жӢҝзқҖеҺҹд»¶пјҢеҺ» AеҢә1еҸ·жһ¶пјҢзңӢеҲ°зҡ„иҮӘз„¶д№ҹжҳҜйҰҷи•үгҖӮ(з”ҹж•Ҳ)

  • ж“ҚдҪң 2пјҡиҝҪеҠ е•Ҷе“Ғ (append) еҮҪж•°еңЁеӨҚеҚ°д»¶дёҠжҠҠ"иҰҒжӢҝеҮ дёӘе•Ҷе“Ғ"д»Һ 3 ж”№жҲҗдәҶ 4пјҢе№¶еҫҖжһ¶еӯҗдёҠеӨҡж”ҫдәҶдёҖдёӘжўЁгҖӮ дҪ жүӢйҮҢзҡ„еҺҹд»¶пјҢдёҠйқўеҶҷзҡ„иҝҳжҳҜ"иҰҒжӢҝ 3 дёӘ"гҖӮ еҪ“дҪ еҺ»д»“еә“еҸ–иҙ§ж—¶пјҢдҪ еҸӘжӢҝеүҚ 3 дёӘпјҢе®Ңе…ЁдёҚзҹҘйҒ“з¬¬ 4 дёӘжўЁзҡ„еӯҳеңЁгҖӮ(дёҚз”ҹж•Ҳ)

жҖҺд№Ҳи§ЈеҶіпјҹвҖӢ

еҰӮжһңдҪ еёҢжңӣ append д№ҹиғҪиў«еӨ–йқўзңӢеҲ°пјҢеҝ…йЎ»жҠҠ**жӣҙж–°еҗҺзҡ„жё…еҚ•пјҲеҲҮзүҮеӨҙпјү**дј еӣһз»ҷеӨ–йқўгҖӮ

ж–№жі• 1пјҡиҝ”еӣһж–°еҲҮзүҮпјҲжңҖеёёз”ЁпјүвҖӢ

func addOne(s []int) []int {
    s = append(s, 100)
    return s // жҠҠдҝ®ж”№дәҶ Len зҡ„ж–°еҲҮзүҮеӨҙдј еҮәеҺ»
}

// и°ғз”Ё
list = addOne(list) // еӨ–йғЁз”Ёж–°еҲҮзүҮиҰҶзӣ–ж—§еҲҮзүҮ

ж–№жі• 2пјҡдј еҲҮзүҮзҡ„жҢҮй’Ҳ (Pointer to Slice)вҖӢ

иҝҷе°ұзӣёеҪ“дәҺжҠҠ"жё…еҚ•зҡ„еҺҹд»¶"дј иҝӣеҺ»дәҶпјҢиҖҢдёҚжҳҜеӨҚеҚ°д»¶гҖӮ

func addOne(s *[]int) {
    *s = append(*s, 100) // зӣҙжҺҘдҝ®ж”№еӨ–йғЁйӮЈдёӘеҲҮзүҮеӨҙзҡ„ Len е’Ң Ptr
}

// и°ғз”Ё
addOne(&list)
