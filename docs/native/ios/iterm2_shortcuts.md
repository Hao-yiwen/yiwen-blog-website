# iTerm2 快捷键速查表

iTerm2 是 macOS 上最强大的终端替代品。提到"iTerm 常用命令"，通常包含两层含义：

1. **iTerm2 软件本身的快捷键**（如分屏、搜索、多窗口操作）。
2. **通用 Shell 命令**（如 `cd`, `ls` 等，这些在自带终端里也通用）。

iTerm2 真正的核心优势在于**快捷键和分屏管理**。以下是按照使用场景整理的常用操作清单。

---

## 1. 分屏与窗口管理 (最核心功能)

这是 iTerm2 相比系统自带终端最大的优势，可以在一个窗口内同时查看多个任务。

| 功能 | 快捷键 | 说明 |
| --- | --- | --- |
| **垂直分屏** | `Cmd + d` | 左右并列两个窗格 |
| **水平分屏** | `Cmd + Shift + d` | 上下并列两个窗格 |
| **切换焦点** | `Cmd + Option + 方向键` | 在不同分屏之间光标移动 |
| **关闭当前分屏** | `Cmd + w` | 关闭当前光标所在的窗格 |
| **最大化当前分屏** | `Cmd + Shift + Enter` | 让当前窗格全屏（再按一次恢复） |
| **调整分屏大小** | `Ctrl + Cmd + 方向键` | 调整当前窗格的宽/高 |

## 2. 标签页 (Tab) 管理

当你不想在一个屏幕挤太多分屏时，可以使用多标签页。

| 功能 | 快捷键 | 说明 |
| --- | --- | --- |
| **新建标签页** | `Cmd + t` | 打开一个新的 Tab |
| **切换标签页** | `Cmd + 数字` | 切换到第 N 个标签页 (如 `Cmd + 1`) |
| **左右切换** | `Cmd + Shift + [` 或 `]` | 循环切换前一个/后一个标签页 |

## 3. 高级功能 (iTerm2 独有)

这些功能是 iTerm2 的"杀手锏"，能极大提升效率。

### 剪贴板历史 (`Cmd + Shift + h`)

iTerm2 自带剪贴板历史记录。按下快捷键后，会弹出一个列表，显示你最近复制/粘贴过的所有文本，支持搜索和直接回车粘贴。

### 广播输入 (`Cmd + Option + i`)

**非常强大！** 当你开启此功能，你在当前键盘输入的内容，会**同时发送到当前 Tab 下的所有分屏中**。

**场景：** 你需要在 3 台服务器上同时执行 `apt-get update`，分屏连上后，开启广播输入，打一次命令即可。

### 智能搜索 (`Cmd + f`)

除了普通的查找，iTerm2 的搜索栏支持正则表达式。按下 `Tab` 键可以自动向右扩展选中的词。

### 即时回放 (Instant Replay) (`Cmd + Option + b`)

如果你刚才编译刷屏太快，没看清报错，可以按下这个组合键。也就是终端的"时间倒流"功能，可以通过时间轴查看几分钟前的屏幕状态。

## 4. 文本选取与复制

iTerm2 有一套独特的鼠标/文本交互逻辑：

| 操作 | 说明 |
| --- | --- |
| **自动复制** | 鼠标选中即复制（无需按 Cmd+C） |
| **智能选中** | 双击选中单词，三击选中整行 |
| **矩形选中** | 按住 `Option` 键再用鼠标拖拽，可以进行矩形块选择（特别适合复制表格数据） |

---

## 5. 常用 Shell 基础命令 (通用)

无论你用 iTerm 还是系统终端，这些 Zsh/Bash 命令都是必备的：

| 命令 | 说明 |
| --- | --- |
| `ls -G` | 列出当前目录文件（`-G` 开启彩色显示） |
| `cd <路径>` | 进入目录 (`cd ..` 返回上一级，`cd ~` 回首页) |
| `pwd` | 显示当前所在路径 |
| `open .` | **(Mac 独有)** 在 Finder (访达) 窗口中打开当前终端所在的文件夹 |
| `clear` 或 `Cmd + k` | 清除屏幕内容 |
| `history` | 查看输入过的历史命令 |
