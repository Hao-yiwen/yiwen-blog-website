# 打印时候的格式化说明符号

1. 整数类型：

    - %d 或 %i：有符号十进制整数。
    - %u：无符号十进制整数。
    - %x：无符号十六进制整数（小写字母）。
    - %X：无符号十六进制整数（大写字母）。
    - %o：无符号八进制整数。

2. 浮点类型：

    - %f：十进制浮点数（包括小数部分）。
    - %e 或 %E：指数形式的浮点数（小写e或大写E表示指数部分）。
    - %g 或 %G：根据数值的大小，自动选择 %f 或 %e（或 %E）的格式。

3. 字符和字符串类型：

    - %c：单个字符。
    - %s：C 语言风格的字符串（以 \0 结尾的字符数组）。
    - %@：Objective-C 对象类型，通常用于打印 NSString 对象。

4. 指针类型：
    - %p：指针的值（地址）。5. 宽度和精度控制：
    - %m.n：其中 m 是最小字段宽度，n 是小数点后的位数。例如，%6.2f 表示一个至少宽度为6，小数点后有2位的浮点数。
