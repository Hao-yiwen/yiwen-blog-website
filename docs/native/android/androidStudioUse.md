# Android studio使用

因为大家熟知的原因，无法正常访问谷歌资源，所以此时需要配置一下代理。

## 步骤

1. 点击`Android studio`打开`Settings`
2. 在`Apperance & Behavior`下面`System Settings`侧打开`Http Proxy`
3. 使用魔法配置代理
